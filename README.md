## What is EKA?

**EKA** is a next-generation AI chatbot built to deliver intelligent, conversational experiences right in your browser. Evolved from the earlier **Swastik AI** project, EKA represents a reimagined approach to human–AI interaction — minimal, fast, and purposeful.

🌐 **Live Demo:** [https://Abhiraj1121.github.io/ai/](https://Abhiraj1121.github.io/ai/)

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- No installation required for the live version

### Running Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/Abhiraj1121/ai.git
   cd ai
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your browser, or use a local server:
   npx serve .
   # Then visit http://localhost:3000
   ```
